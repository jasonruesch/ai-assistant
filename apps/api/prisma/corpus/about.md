# About Jason Ruesch

Jason Ruesch is a frontend-focused full-stack software engineer. His personal
site is a fast, accessible portfolio built as a React 19 + Vite single-page
app with file-based routing, a Tailwind v4 design system, light/dark theming,
and PWA support, deployed to GitHub Pages via GitHub Actions.

The portfolio demonstrates a consistent engineering philosophy across every
project: accessibility first (WCAG-compliant components, keyboard navigation,
focus management), a shared design system so every app looks and behaves
consistently, strong typing end to end with TypeScript, and real deployments
rather than mockups wherever it matters.

SourceSage is one of those projects: an LLM-powered application that
wraps Anthropic's Claude in a fast, accessible chat interface built on the same
design system, with streaming responses, tool use, and retrieval-augmented
context (RAG) over this very knowledge base.
